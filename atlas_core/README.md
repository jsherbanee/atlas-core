# Atlas Core

Atlas Core is a small scaffold for building Python projects with domain models, parsers, services, and a CLI.

## Structure

- `atlas_core/cli` - command-line entry points
- `atlas_core/domain` - domain models and entities
- `atlas_core/parsers` - parsing utilities
- `atlas_core/services` - service layer implementations
- `atlas_core/utils` - reusable helpers

## Quick start

```bash
python -m atlas_core.cli --version
```
